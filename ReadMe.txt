# It is small app that is Created for learning NodeJs, MongoDb, ExpressJs.
# It's name is TodoApp, in this we can :
# create task
# delete existing task
# update existing task
# enjoy using this app and have fun :-)

To use this at your system:
# You need to clone this file into a directory (let say 'dir')
# Open terminal and come at this directory
# Now first type command npm init
# Then install 3 packages as mention below :
    "body-parser": "^1.19.0",
    "express": "^4.17.1",
    "mongodb": "^3.3.2"
# first start the mongo db in your system then start this app by using "node app.js"
